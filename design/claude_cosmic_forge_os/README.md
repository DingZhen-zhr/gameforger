# Claude Cosmic Forge OS Design Drop

This folder contains the Claude Design handoff imported for the GameForger visual refresh.

## Files

- `GameForger Cosmic Forge OS - v1.html`: assembled preview.
- `forge-components.jsx`: shared React visual components and design tokens.
- `screens-tabs.jsx`: Projects, Gallery, and Control tab designs.
- `screens-flow.jsx`: Splash, Workspace, and Preview flow screens.
- `design-canvas.jsx`, `ios-frame.jsx`: presentation shell helpers.
- `all-screens.jpg`, `overview.jpg`, `zoomed.jpg`: rendered references.

## Flutter Mapping

- Black cosmic shell: `CosmicBackground`.
- Liquid glass cards and rows: `ForgeGlassCard`.
- Brand mark and project thumbnails: `NebulaOrb`, `NebulaSeed`.
- Status pills and dimension bars: `ForgeChip`, `ForgeEnergyBar`.
- Floating controls: `ForgeIconButton`, `ForgePrimaryButton`.

The implementation keeps existing Flutter routes, providers, and business logic intact; this drop is used as a visual language reference.
